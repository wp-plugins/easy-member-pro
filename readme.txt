== Description ==

An Extension to Easy Member Pro.
Wp-EasyMemberPro is a wordpress plugin that allows you to protect your posts, ensuring that only your logged in members, with the proper membership level, and drip feed options can view them.

== Installation ==

1. Upload the whole wp-EasyMemberPro folder to your wp-content/plugins/ folder
2. Go to your Wordpress admin -> Plugins and activate the wp-EasyMemberPro plugin.
3. Goto Settings -> WP EasyMemberPro -> Add the URL to your EasyMemberPro installation & update other options and press Save Changes;
3. That should be it.
